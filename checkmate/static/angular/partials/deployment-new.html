Heads up! You need to select an environment and blueprint before I can show you the settings.
Settings
  • Help
  • This screen allows you to create and launch a deployment. You'll start by selecting an environment and a blueprint. Both of these selections will expose settings that you will need to configure how you want. Once you are happy with everything, click the Build It! button and Checkmate will start provisioning your environment.
  • Simulation
  • Checkmate will allow you to run a simulation based on the deployment you have selected. This is mostly used for debugging purposes.
